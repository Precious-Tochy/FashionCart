Ify's Fashion Wears

Overview:
Ify's Fashion Wears is a responsive e-commerce website built with Laravel. It allows users to browse, search, and shop for fashion items including dresses, gowns, tops, jackets, and knits. The website supports dynamic categories, subcategories, and personalized features such as wishlists and shopping bags.

The design is mobile-friendly, with tailored navigation for desktop, tablet, and mobile views. Users can filter products, view trending items, and explore seasonal collections.

Features:
Fully responsive design (desktop, tablet, mobile)
Product search functionality
Category and subcategory navigation
Wishlist and shopping bag with live cart count
Carousel sliders for promotions and featured collections
Product showcase sections: Trending, Limited Edition, Event Season
Dynamic Laravel backend with database-driven content
Integration with Bootstrap 5 and AOS animation library
Styled using custom CSS and Remix Icons
Multi-device navigation support (desktop nav, tablet bar, mobile scroll)

Technologies Used:
Backend: Laravel PHP Framework
Frontend: HTML, CSS, JavaScript
Styling & Layout: Bootstrap 5, Custom CSS
Icons: Remix Icons
Animations: AOS (Animate On Scroll)
Database: MySQL (via Laravel Eloquent ORM)
Version Control: Git & GitHub
